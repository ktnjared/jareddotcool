Jared Schmidt

Headphones and hoodies. Retro enthusiast (Apple, IBM, video games, computers).

If you are reading this, "Hi!".

I wrote my first personal webpage in November 1994, hosted on my high school's web server at http://tjhs.fr.k12.md.us/.

That web page moved to http://www.kis.net/dragon in 1996.

After that, I bought my first domain in 1998, dragon-rider.org, and began the journey of running my own servers, first a co-lo'd Windows 98 computer with the FrontPage 98 web server, eventually moving to Apache Web Server under Red Hat Linux (not to be confused with RHEL).

This web page is just me having fun and making a personal web page that will work well with retro computers, and eventually, lynx.
